# WhatsappScripts

Código para enviar o Script inteiro de Shrek ou ~~Bee Movie~~ (falta implementar) no whatsapp web

A proposta desse repositório é manter ele mais atualizado, visto que o primeiro e original esta obsoleto

## Utilização

Abra o console do seu navegador e digite `allow pasting`

Copie todo o conteúdo do arquivo shrek.js (clique em raw -> ctrl+a -> ctrl+c, ou apenas clique no botão de copiar)

Abra a conversa desejada, cole o código no console e aperte Enter
